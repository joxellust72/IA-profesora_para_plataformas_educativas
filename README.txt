EL USO QUE SE LE DE A LA IA ES DE ESTRICTA RESPONSABILIDAD DEL USUARIO

------------------Pasos para ejecutar el programa:------------------
1. Instalar Python desde la página oficial: https://www.python.org/downloads/
   Asegúrate de marcar la casilla "Add Python to PATH" durante la instalación.

2. Ejecutar el servidor de voz (TTS).
   - Haz doble clic en el archivo `run_tts_server.bat`.
   - La primera vez, se instalarán las dependencias y se descargará el modelo de voz. Este proceso puede tardar varios minutos.
   - Espera a que aparezca el mensaje "¡Modelo cargado y listo!".
   - **Importante:** No cierres esta ventana de terminal mientras usas la aplicación.

3. Ejecutar el servidor web principal.
   - Haz doble clic en el archivo `ejecutarServer.bat`.
   - Esto iniciará el servidor y abrirá automáticamente tu navegador en la dirección http://localhost:8000.
   - El modelo de IA tardará un momento en cargar en el navegador. Podrás ver el progreso en la caja de chat.

------------------Consideraciones adicionales------------------
- Para usar el reconocimiento de voz (el botón del micrófono), debes utilizar un navegador compatible como Google Chrome o Microsoft Edge.
- La aplicación requiere una conexión a internet para descargar los modelos la primera vez.
- El rendimiento del modelo de IA dependerá de la capacidad de tu computador.
